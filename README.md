# Insania Web
Проект веб-приложения проекта "Инсания"

## Стандарты кода
* Расположение:
	* display - тип отображения
	* flex-direction - направление дочерних элементов
* Отступы:
	* margin - внешние отступы
* Размеры:
	* height - высота
* Внешний вид
	* color - цвет 
* Шрифт
	* font-family - семейство шрифта
	* font-size - размер шрифта
